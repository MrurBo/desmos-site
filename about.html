About Desmos Graph Collection

What is this?

This is a curated collection of interactive mathematical visualizations created using Desmos. Our goal is to showcase beautiful, educational, and interesting mathematical concepts through interactive graphs.

How to Use

  • Browse: Use the search bar and tag filters to find graphs you're interested in
  • Preview: Each graph shows a preview of what you'll see
  • Interact: Click "Open in Desmos" to explore the full interactive version

Contributing

Want to add your own graphs to the collection? Here's how:

  1. Create your graph on Desmos
  2. Make sure your graph is saved and you have a shareable link
  3. Fork this project on GitHub
  4. Add your graph information via python cli.py add
  5. Submit a pull request

Graph Requirements

  • Graphs must be original or properly credited
  • Include a clear title and description
  • Add relevant tags for easy discovery
  • All graphs, geometry, and 3d graphs must have their respective tags.
  • Make sure the graph is well-documented and easy to understand
